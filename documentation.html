Documentation

For the 2.0 version, the library has been almost entirely rewritten with some major changes. You'll find here everything you'll need to stay up to date with the latest improvements.

Installation

In a browser: 

<script src="curtains.min.js"></script>
Using npm: 

npm i curtainsjs
Load ES module: 

import {Curtains} from 'curtainsjs';
